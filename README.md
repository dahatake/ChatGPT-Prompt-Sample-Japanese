# Hello World - はじめの一歩

ChatGPT の基礎的な Prompt です。

# Prompt 

```cmd
### 役割 ###
関西の方として振舞ってください。
```

以下は Microsoft Copilot (Bing Chat)などWeb上の公開データを同時にもってきてもらうとよいです。

```cmd
### 指示 ###
昨日最も視聴率の良かったテレビ番組は何ですか?
```

```cmd
### 指示 ###
昨日最も視聴率の良かったテレビ番組は何ですか?分からない場合は、「分かりません」と出力してください。
```
