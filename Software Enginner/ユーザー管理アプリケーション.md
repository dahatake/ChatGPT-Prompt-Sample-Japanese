# Prompt

```cmd
### 役割 ###
上級ITコンサルタントとして振舞ってください。

### タスクの実行 ###
以下のビジネス上の課題があります。ITを活用して試作品を開発するためには、どんな手順がありますか?表形式で出力してください。分からない場合は「分からない」と出力してください。

ビジネス上の課題:"""
- ビジネス圏が[xxx県]の県域に限る
- 人口が減っている
- 1ユーザーあたりの売り上げを望めない。追加商品が開発出来ていない
- 新規ユーザー獲得のために、インターネットをチャネルとして活用したい
- ECサイトは作成した。Rakutenにも出展している。
- それでもユーザーが増えていない
- ユーザーからのフィードバックとして「なぜ個別にユーザー登録をするのか?面倒だ」が上がっている
- ECサイトへのログイン時に、ユーザーが既に持っているFacebook/Twitterなどの既存アカウントを使ったらいいのではないか?という意見がある
- 社内にITに詳しい人がいない
"""
```

```cmd
### タスクの実行 ###
素晴らしいです。
外部とやりとりするために、ドキュメントを作りたいです。先ほどの「ビジネス上の課題」から、要件定義のドキュメントを作成してください。
```

```cmd
### タスクの実行 ###
このアプリケーションを Microsoft Azure 上で動かします。概算の見積もりをするためには、どんな情報が必要ですか? 分からなければ「わかりません」と出力してください。
```

```cmd
### タスクの実行 ###
今回指定している、以下の、ビジネス上の課題。を解決するアプリケーションをするうためには、どんなAzureの製品が必要ですか?リスト形式で出力してください。分からなかったら、「分からない」と答えてください。

ビジネス上の課題:"""
- ビジネス圏が[xxx県]の県域に限る
- 人口が減っている
- 1ユーザーあたりの売り上げを望めない。追加商品が開発出来ていない
- 新規ユーザー獲得のために、インターネットをチャネルとして活用したい
- ECサイトは作成した。Rakutenにも出展している。
- それでもユーザーが増えていない
- ユーザーからのフィードバックとして「なぜ個別にユーザー登録をするのか?面倒だ」が上がっている
- ECサイトへのログイン時に、ユーザーが既に持っているFacebook/Twitterなどの既存アカウントを使ったらいいのではないか?という意見がある
- 社内にITに詳しい人がいない
"""
```

```cmd
### タスク実行 ###
[サービス名] の概算見積もりに必要な情報は何ですか? 今回のビジネス上の課題を前提にして教えてください。また、その単価の情報から、今回のビジネス上の課題を元にして必要となる容量を算出して、概算見積もりを行ってください。作成結果は、表形式で出力してください。分からなかったら「分からない」と答えてください。
```

